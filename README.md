## 在网页上引入Live2D看板娘

1.将项目解压并放入web容器或者使用Nginx或Apache等工具反向代理来请求模型资源
 - 没测试过本地请求是否成功

2.修改你的资源请求路径
 - 模型默认可放在Resource下

3.打开网站运行显示
 - 如果出现跨域问题，请最好本地调试
 - 本Demo可以配合大佬写的waifu.js来实现其他功能 

代码命名注释等不规范会慢慢修改
